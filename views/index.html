The web page is using data from FDA Adverse Event Reporting System (FAERS). On the search page you can search drugs by name or manufacturer, after that you can select a drug and the web displays advers event occured on people who has used the selected drug. This page was created as a continuance of a school project using JAVA, node.js (some frameworks), Heroku (hosting the web page), Google Cloud Platform (hosting PostgreSQL database), Bootstrap.

I am using one year free hosting on Heroku and GCP (from august 2017) and imported only a small amount of data.
